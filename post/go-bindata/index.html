Go打包静态资源和配置文件#

场景：#

需要将配置文件打包类似和java的application.yml配置文件打包到最终的jar包内。

打包工具go-bindata#

下载#

  1. 下载

    # 这种下载方式可以使用，go get 方式不行
go install github.com/go-bindata/go-bindata/go-bindata@latest

  2. 测试

    go-bindata

常用操作#

打包配置文件#

测试项目目录层级

需求：打包storeInfo.yml配置文件

go-bindata .\config\storeinfo.yml


# 修改默认生成的文件名和包名，必须有双引号
go-bindata -o=".\asset\asset.go" -pkg="asset" .\config\storeinfo.yml

读取文件

# 读取文件修改为如下方式
asset.Asset("config/storeInfo.yml")
