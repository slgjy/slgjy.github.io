GoLand-Swagger#

swagger简介#

根据代码中自定义注解生成接口文档，在前后端分离项目中通过swagger将接口文档定义好，自动更新，方便维护。

swagger集成#

依赖下载#

go install github.com/swaggo/swag/cmd/swag@latest
go get -u github.com/swaggo/gin-swagger
go get -u github.com/swaggo/files

初始化#

#swag初始化
swag init

在项目目录下生成如下目录

引入生成的包#

import (
	_ "statistical_data/docs"  //就是swag init生成的包
)

// @title 项目api文档
// @version 1.0
// @description API文档
// @host 127.0.0.1:8199
// @BasePath /
func main() {
    ....
}

指定相关选项生成

swag init -g main.go -d apidir --parseDependency -o docs

重新初始化

swag init

测试#

访问地址：http://ip:port/swagger/index.html

接口定义#

请求示例1#

// UserListView 用户列表
// @Tags 用户管理
// @Summary 用户列表查询
// @Description 查询用户列表
// @Param data query UserListRequest true "参数"
// @Param token header string true "token"
// @Router /api/user [get]
// @Produce json
// @Success 200 {object} res.Response{data=res.ListResponse[models.UserModel]}
func (UserApi) UserListView(c *gin.Context) {
	var cr UserListRequest
	c.ShouldBindQuery(&cr)
	var user []models.UserModel
	var count int
	offset := (cr.Page - 1) * cr.Limit
	global.DB.Model(models.UserModel{}).Select("count(id)").Scan(&count)
	global.DB.Limit(cr.Limit).Offset(offset).Find(&user)
	res.OkWithList(user, count, c)
	return
}

文件上传示例#

// ImageUploaderView 上传图片
// @Tags 图片管理
// @Summary 上传图片
// @Description 上传图片
// @Param image formData file true "文件上传"
// @Param token header string true "token"
// @Router /api/image [post]
// @Accept multipart/form-data
// @Produce json
// @Success 200 {object} res.Response{}
func (ImageApi) ImageUploaderView(c *gin.Context) {
	file, err := c.FormFile("image")
	if err != nil {
		res.FailWithMsg("上传参数错误", c)
	}
	fmt.Println(file.Filename)
	res.OkWithMsg("图片上传成功", c)
}

注释含义#

Tags#

用于给api分组

Produce#

返回数据结构，一般采用json格式

Param#

参数格式

@Param 1.参数名 2.参数类型 3.参数数据类型 4.是否必须 5.参数描述 6.其他属性

参数类型#

参数类型指代前端传递参数的形式。

  • header
  • body
  • query
  • path
  • formData

参数数据类型#

  • sring
  • integer
  • number
  • user defined struct（自定义结构体）

Success#

指定成功的响应数据格式

@Success 1.http响应码 {2.响应参数类型} 3.响应数据类型 4.其他描述

Router#

@Router /path/to/handle [http方法]
