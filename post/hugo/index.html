hugo#

hugo是由go语言实现的静态网站生成器。

hugo下载#

  1. 下载地址

    https://github.com/gohugoio/hugo/releases

  2. 下载后解压

hugo常用命令#

  1. 新建站点

    hugo new site myblog

  2. 启动

    hugo server -D

  3. 设置主题

    # 地址
https://github.com/gohugoio/hugo/releases

    1. 在生成站点的目录下的themes下克隆

    2. 采用PaperMod主题

    3. 使用指定主题启动

      #修改配置hugo的配置文件
#hugo部署站点(github仓库)
baseURL = 'http://slgjy.github.io'
languageCode = 'en-us'
#博客名称
title = '小石头博客'
#主题信息
theme = 'hugo-PaperMod'

  4. 创建博客文档

    #其会在博客目录content 生成post/文档名
hugo new post/go单元测试.md

hugo部署到github#

  1. 新建github仓库

    仓库名称必须是xxx.github.io，xxx替换为自己的github名称

  2. 将该地址设置到站点配置文件

  3. 同步public的文件到仓库地址

    cd public
git init
git add .
git commit -m 'init'
git remote add origin <仓库地址>
git push -u origin master

  4. 测试一下
