Jasypt加密操作

May 11, 2023

概述#

Jasypt是一个加密库，使用敏感信息数据的加密。比如springboot项目中配置文件的明文密码数据；数据信息中的敏感数据加密。

使用#

依赖引入#

<dependency>
    <groupId>com.github.ulisesbocchio</groupId>
    <artifactId>jasypt-spring-boot-starter</artifactId>
    <version>3.0.5</version>
</dependency>

简单加密#

加密口令#

==秘钥安全：==

为了安全设置，通常采用启动时添加启动参数(环境变量)的方式来配置秘钥

-Djasypt.encryptor.password=test

jasypt:
  encryptor:
    # 配置的秘钥(加盐)
    password: test
    # 指定界面算法，需要和加密使用的一致
    algorithm: PBEWithMD5AndDES
    # 指定initialization vector类型
    iv-generator-classname: org.jasypt.iv.RandomIvGenerator
    # 配置前后缀默认是"ENC()"
    property:
      prefix: "ENC("
      suffix: ")"

测试(获取密文)#

@SpringBootTest
@RunWith(SpringRunner.class)
public class MainAppTest {

    @Autowired
    private StringEncryptor stringEncryptor;

    @Test
    public void testJasypt() {
        // password需要加密的明文密码
        String enpassword = stringEncryptor.encrypt("password");
        System.out.println("encode password:" + enpassword);
        System.out.println("decode password:" + stringEncryptor.decrypt(enpassword));
    }
}

spring:

    name: changan-system

    nacos:

        username: nacos

        password: ENC(xxx)

自定义加密类#

上述采用的加密算法为PEB算法，根据我们的业务需求也可以自定义加密类。

自定义加密类实现StringEncrypto接口，并在配置文件中指定

@Component
public class Encryptor implements StringEncryptor{

    @Override
    public String encrypt(String s) {
        // 加密逻辑
        return xxx;
    }
    @Override
    public String decrypt(String s) {
        // 解密逻辑
        return xxx;
    }
}

指定加解密算法

jasypt:
  encryptor:
    # 指定加密算法后，不在需要使用口令进行加解密了
    # password: test
    # 指定加密算法
    bean: encryptor

非对称加密#

生成公钥与私钥借助了工具类库Hutool，依赖如下

<dependency>
    <groupId>cn.hutool</groupId>

    <version>5.4.3</version>
</dependency>
<dependency>
    <groupId>org.bouncycastle</groupId>
    <artifactId>bcprov-jdk15on</artifactId>
    <version>1.66</version>
</dependency>

KeyPair pair = SecureUtil.generateKeyPair("RSA");
System.out.println(Base64.getEncoder().encodeToString(pair.getPublic().getEncoded()));
