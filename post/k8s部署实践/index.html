部署Mysql服务#

创建pv#

PersistentVolume(简称pv)持久卷，也是一种集群资源，他的声明周期和使用他的pod项目独立

新建mysql-pv.yaml

apiVersion: v1
kind: PersistentVolume
metadata:
  namespace: app-storage
  name: mysql-pv
  labels:
    pv: mysql-pv
spec:
  capacity:
    storage: 10Gi
  accessModes:
  - ReadWriteOnce
  persistentVolumeReclaimPolicy: Retain
  hostPath:
    path: /root/data

# 创建PV
kubectl create -f mysql-pv.yaml
# 检验
kubectl get pv
创建pvc#

PersistentVolumeClaim（持久卷声明，简称PVC）是用户的一种存储请求。它和Pod类似，Pod消耗Node资源，而PVC消耗PV资源。Pod能够请求特定的资源（如CPU和内存）。PVC能够请求指定的大小和访问的模式（可以被映射为一次读写或者多次只读）。

新建mysql-pvc.yaml

apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: mysql-pvc
  namespace: app-storage
spec:
  storageClassName: ""
  accessModes:
    - ReadWriteOnce
  resources:
    requests:
      storage: 8Gi
  selector:
    matchLabels:
      pv: mysql-pv

# 创建 PVC
kubectl create -f mysql-pvc.yaml
# 校验
kubectl get pvc
创建Deployment#

新建mysql-deploy.yaml

apiVersion: apps/v1
kind: Deployment
metadata:
  name: mysql-deploy
  namespace: app-storage
spec:
  replicas: 1
  selector:
    matchLabels:
      name: mysql-ops
  template:
    metadata:
      labels:
        name: mysql-ops
    spec:
      containers:
        # 容器名称
        - name: mysql
          #镜像 这里采用自己的私有仓库ip地址
          image: ip:15000/library/mysql:5.7.18
          # 镜像拉取规则
          imagePullPolicy: IfNotPresent
          env:
          - name: MYSQL_ROOT_PASSWORD
            # 自定义数据库密码(设置为自己的)
            value: "xxxxx"
          - name: TZ
            value: Asia/Shanghai
          ports:
            - containerPort: 3306
          volumeMounts:
            - name: mysql-persistent-storage
              mountPath: "/var/lib/mysql"
      #和申请的pvc映射
      volumes:
        - name: mysql-persistent-storage
          persistentVolumeClaim:
            claimName: mysql-pvc

# 创建 Deployment
kubectl create -f mysql-deploy.yaml
# 校验
kubectl get deploy
创建 Service#

新建mysql-service.yaml文件

apiVersion: v1
kind: Service
metadata:
  name: mysql-svc
  namespace: app-storage
  labels:
    name: mysql-svc
spec:
  type: NodePort
  ports:
  - port: 3306
    protocol: TCP
    targetPort: 3306
    name: http
    nodePort: 30306
  selector:
    name: mysql-ops

# 创建 Service
kubectl create -f mysql-service.yaml
# 校验
kubectl get svc

  name: vue #pod节点名称
  namespace: app-storage
  labels:
    app: changanstorageweb
spec:
  replicas: 1
  selector:
    matchLabels:
      app: changanstorageweb    # has to match .spec.template.metadata.labels
  template:
    metadata:
      labels:
        app: changanstorageweb # has to match .spec.selector.matchLabels
    spec:
      imagePullSecrets:
      - name: harbor-secret
      containers:
      - name: changanstorageweb
        image: 10.62.4.92:15000/library/changanstorageweb:v1.0 #pull镜像

            cpu: 400m
